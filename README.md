Repositório das **notas de aula, materiais e aplicações computacionais da disciplina de Estatística II**, oferecida na **Universidade de São Paulo (USP)**.

O objetivo deste repositório é organizar de forma **reprodutível e transparente**:

- materiais didáticos  
- notas teóricas  
- exemplos computacionais  
- bases de dados utilizadas nas aulas  
- exercícios e aplicações empíricas  

Todo o conteúdo é estruturado utilizando **Quarto**, permitindo gerar **um site público com as notas da disciplina**.

# 📌 Objetivos do Repositório

Este projeto tem como finalidade:

- Organizar **notas de aula de Estatística II** de forma estruturada  
- Desenvolver **exemplos empíricos e computacionais** aplicados à análise de dados  
- Integrar **teoria estatística e aplicações práticas** em R e Python  
- Garantir **reprodutibilidade das análises**  
- Disponibilizar um **site acadêmico aberto** com os materiais da disciplina  

---

Este repositório pode ser utilizado em colaboração com **alunos ou assistentes da disciplina**.

---

# ⚠️ Observação Importante

Alguns arquivos e pastas são **necessários para o funcionamento do Quarto e do GitHub Pages** e não devem ser alterados ou movidos da raiz do projeto.

Devem ser modificados apenas:

- arquivos `.qmd`
- conteúdos das pastas de materiais (`AULAS`, `EXERCICIOS`, `DADOS`, etc.)
- documentação do projeto.
